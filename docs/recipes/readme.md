# Recipes

Step-by-step guides for building common Django features with trim.

## Getting Started

+ [Building a Contact Form](./basic-page.md) - Your first complete feature: model, form, view, and URLs
+ [Building a ListView](./listview.md) - Display data in tables with filtering and pagination
+ [Todo List](./todo-list.md) - Build a simple task management app

## What You'll Learn

These recipes show you how to:

- Structure Django apps following the Model → View → URL pattern
- Use trim's shortcuts to write less boilerplate
- Build forms that validate and save data
- Display lists of data with class-based views
- Wire everything together with clean URLs

Each recipe is designed to teach core Django concepts while showing how trim makes your code cleaner and faster to write.

**New to Django?** Start with [Building a Contact Form](./basic-page.md) - it covers the fundamentals.
